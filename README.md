# 这是一个obsidian插件
![[Pasted image 20260329152810.png|350]]
## 主要功能说明
### 1. 三种创建模式

树形结构模式：支持多级嵌套，使用缩进表示层级

简单文件夹模式：逗号分隔，创建一级文件夹

简单文件模式：逗号分隔，创建一级md文件

### 2. 树形结构解析规则
支持2或4个空格的缩进，也支持Tab键

以.md结尾或包含.的名称视为文件

无扩展名视为文件夹

空行自动忽略
